## üì± UI/UX Rules
1. **Mobile-First**: Crypto moves fast; the UI must be perfect on a phone.
2. **Low Cognitive Load**: Don't show all data at once. Use progressive disclosure.
3. **Agent-Centric**: The UI should reflect that an AI is "thinking" and "executing" in the background (using subtle pulse animations).
4. **Security Visibility**: Security shouldn't just be a box; it should be visible (e.g., a "Safe" shield icon that glows when the audit passes).
