- **Psychology:** Creative, connective, ecosystem-focused

## Psychology Behind Each Color

### Deep Indigo (#2d1b69)
- **Associations:** Wisdom, mystery, premium crypto
- **Why it works:** Dark enough for contrast, purple undertone feels premium
- **Usage:** Primary brand color, headers, navigation

### Electric Coral (#ff4d6d)  
- **Associations:** Action, energy, attention
- **Why it works:** Warm red-pink creates urgency without aggression
- **Usage:** Primary CTAs, important alerts, action buttons

### Cyber Lime (#00ff9d)
- **Associations:** Growth, success, innovation
- **Why it works:** Bright green signals positive movement (profits)
- **Usage:** Success states, profit indicators, growth charts

### Gold (#ffd166)
- **Associations:** Wealth, value, premium
- **Why it works:** Literal "gold" for wealth management
- **Usage:** Premium features, wealth highlights, VIP indicators

## Implementation in UI

### Dashboard Example:
- **Background:** Slate Gray (#2a2d3e)
- **Cards:** Deep Indigo (#2d1b69) with subtle gradient
- **Profit Numbers:** Cyber Lime (#00ff9d)
- **Action Buttons:** Electric Coral (#ff4d6d) with hover ‚Üí Gold (#ffd166)
- **Risk Warnings:** Keep red (#ff4d4d) for universal recognition
- **USDC Indicators:** Keep USDC blue (#2775CA) for consistency

### Accessibility Considerations
- **Contrast Ratios:** All combos meet WCAG AA standards
- **Color Blindness:** Using patterns + color (not color alone)
- **Dark Mode:** Designed for dark-first, with light mode optional
